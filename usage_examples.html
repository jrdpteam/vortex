Vortex v3.0 - Usage Examples

Encrypting a File

        
            python3 vortex.py -f example.txt -e -l 16 -t letter number extra

This command will encrypt the file example.txt using a generated password with a length of 16 characters, including letters, numbers, and special characters (_ and -).

Decrypting a File

        
            python3 vortex.py -f example.txt.vrtx -d

This command will decrypt the encrypted file example.txt.vrtx. You will be prompted to enter the password used during encryption.

Encrypting with a Custom Password

        
            python3 vortex.py -f example.txt -e -c "MyCustomPassword123!"

Encrypt the file example.txt using a custom password MyCustomPassword123!.

Decrypting with a Custom Password

        
            python3 vortex.py -f example.txt.vrtx -d -c "MyCustomPassword123!"

Decrypt the file example.txt.vrtx using the custom password MyCustomPassword123!.

Viewing Help

        
            python3 vortex.py -h

This command displays the help menu with all available options.
